# 🤖 Школьный помощник (Telegram Bot)

Телеграм-бот для автоматизации доступа к школьному расписанию и меню в школьной столовой!

[@schedule_lycey23_bot](https://t.me/schedule_lycey23_bot)

## 🔗 Команды

* `/start` — Начать взаимодействие с ботом.
* `🍔 Меню` — Просмотр молочного и мясного меню (цикл 4 недели).
* `📅 Расписание` — Просмотр основного расписания уроков.
* `📝 Изменённое расписание` — Просмотр оперативных правок в расписании.

## 🛠 Установка и запуск

1. Создайте файл `.env` и напишите в нем токен вашего бота, полученный от @BotFather:

```env
BOT_TOKEN='YOUR_TOKEN'
```

2. Поместите файл ключей доступа Google Cloud creds.json в корневую директорию проекта.

3. Установите необходимые библиотеки:

```Bash
pip install -r requirements.txt
```

4. Запустите бота:

```Bash
python main.py
```
