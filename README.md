# 🤖 Школьный помощник (Telegram Bot)

Телеграм-бот для автоматизации доступа к школьному расписанию и 4-недельному циклическому меню!

[@schedule_lycey23_bot](https://t.me/schedule_lycey23_bot)

## 🔗 Команды

* `/start` — Начать взаимодействие с ботом.
* `🍔 Меню` — Просмотр молочного и мясного меню (цикл 4 недели).
* `📅 Расписание` — Просмотр основного расписания уроков.
* `📝 Изменённое расписание` — Просмотр оперативных правок в расписании.
